# Дополнительные книги для совместного прохождения

Обе книги используют ту же структуру:

- обычные переходы;
- переходы с `answer`;
- локальные тупики;
- общие финалы с `endForAll: true`.

Ниже находятся две готовые книги в JSON. Их можно запустить без изменений или использовать как примеры для собственных историй.
